Lenguaje de Marcas

A markup language is a text-encoding system which specifies the structure and formatting of a document and potentially the relationships among its parts. Markup can control the display of a document or enrich its content to facilitate automated processing.

Why Coose Markup Languages
Human-readable: Markup languages are designed to be read and understood by humans, using standard words and phrases instead of unintelligible syntax used in programming languages.

Tag-based: Markup languages use tags (e.g., < and >) to define elements within a document. These tags are used to annotate the content, making it possible for computers to understand the structure and style of the document.

Structured: Markup languages provide a way to organize and structure content, making it possible to define relationships between different elements.

Extensible: Markup languages, such as XML, allow for the creation of custom tags and objects, making them extensible and adaptable to various domains and applications.

Platform-independent: Markup languages are designed to be platform-independent, meaning they can be used across different operating systems and devices.

Simple syntax: Markup languages typically have a simple syntax, making it easy to learn and use.
Simple to learn: Markup languages are easy to learn, even for non-technical people. They consist of plain text with added tags to define structure and presentation.

Platform-independent: Markup languages are platform-independent, meaning they can be used on any operating system (Windows, Unix/Linux, macOS, etc.) without modification.

Portable: Markup language files are portable, allowing documents to be easily transferred between different systems and devices.

Easy to edit: Markup languages are text-based, making it easy to edit documents using any text editor.

Flexible: Markup languages allow for flexible formatting and presentation, enabling authors to control the look and feel of their content.
Web Development: HTML (Hypertext Markup Language): The core language for structuring content on the web. It defines the elements of a webpage such as headings, paragraphs, images, and links.
< h1 >Welcome to my website< /h1 >
< a href="https://www.LM.com" >Click here< /a >

Document Formatting: XML (Extensible Markup Language): Used for storing and transporting data. It allows for custom tags and is highly flexible. It is used in a wide range of applications such as data interchange and configuration files.
< book >< title >XML for Class< author >Julio Juarez< /author >< /book >

Data Representation and Serialization: JSON (JavaScript Object Notation): While technically a data format, JSON is often used in a markup-like fashion to represent complex data structures in a human-readable way.
{"name": "Julio", "age": 33, "city": "Pardinilla"}
